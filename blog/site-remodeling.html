Site Remodeling
10/18/2015

I learned so much from site remodeling this week, especially bootstrap. Deploying bootstrap is not as easy as I thought since I had a lot of style built into my default CSS file. It got a bit harder to identify the problem when there are multiple style file active. Though the DOM from Chrome is very useful. It shows all CSS and specify which style statement is in effect. All I need to do is to find the one in effect and modify or remove the style to allow the Bootstrap takes the effect.

Searching online about Bootstrap tricks is the most fun but time consuming task. It is very easy to get distracted to an interesting topic forgetting what I was looking for. Time-boxing helps a lot on this scenario.

All in all, I change the color code and add some visual effect on the site. It would be great if you can give me some feedback.
